# Re-cipes

Tool to search for recipes based on your current state of fridge and to 
optimize usage of resources.


## How to run 

Prerequisites:
 - docker-compose
 - docker
 - node 14.19.3 

To run backend:

In root directory:
```bash
docker-compose up
```

to run frontend:

```bash
cd client
npm install
npm run dev
```
